Packages.

Here is a list of all the packages that we can provide, all packages include hosting and domain registration.

Package 1.

This package includes a professionally designed web site, built by one of our designers using a service to rapidly create your site. This is to get you up and running as quickly as possible.

This can be a simple information site, a larger web site with multiple pages including photographs, testimonials, or customer feedback. Or it could be a basic e-commerce site, offering a catalogue of the business's products which can be browsed and ordered via the web site.

We will work with you to show you how to use and update your new website, best practices for using.

Package 2.

This will offer bespoke designs for complex web sites which will require elements that would take longer to develop. They will be programmed according to each client's needs and will be offered for more labour intensive projects. It could again be made to provide any particular type of site which is required.

It will also include:

  • Search engine training.
  • Hand made websites.
  • Crafted to fit needs.
  • Almost endless options for customisation.
  • Hosting and Domains.

Package 3.

This package uses the same development tools as package 1 to get you or your organisation up and running with a web presence as soon as possible. But here's the twist, once the website is operational we will the begin development on your brand new custom website. Once this site is completed and tested we will replace the basic site we originally created for you with your brand new site.

It will also include:

  • Search engine training.
  • Hand made websites.
  • Crafted to fit your needs.
  • Almost endless options for customisation.
  • Hosting and Domains.
  • A web presence very quickly replaced by a bespoke site designed to your specifications.
